如何部署博客到Github Pages

  • 注册空仓库，例如：

https://github.com/yumoz/blogs

  • 克隆到本地，下载mkdocs工具；
# 下载mkdocs工具
pip install mkdocs
# 下载完成查看版本
CMD中输入命令：mkdocs --version
mkdocs, version 1.6.0 from C:\Users\*****
  • 修改索引文件

修改mkdocs.yml大致格式如下

site_name: yumoz博客系统
site_url: https://www.gitee.com/yumoz 
nav:
    - 首页: index.md
    - 关于: about.md
theme: 
    name: mkdocs
    highlightjs: true
    hljs_languages:
        - yaml
        - rust
        - python
        - cpp

    locale: zh_CN

    markdown_extentions:
      - admonition

    plugins:
      - search

    extra_css:
      - resources/css/extra.css

进入写文档的页面

mkdocs gh-deploy
